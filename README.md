# Bernardo Biazus — Portfólio

Portfólio pessoal de Bernardo Biazus, Full Stack Developer & UI/UX Designer.

Conceito visual "Carbon/Ember": dark cinematográfico, quase monocromático, com acento laranja-brasa usado com parcimônia.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS v4 (tokens via `@theme`)
- React Three Fiber + drei (hero com campo de partículas em GLSL/curl noise)
- GSAP + ScrollTrigger + SplitText (via `useGSAP`)
- Lenis (smooth scroll, sincronizado ao ticker do GSAP)
- next-intl (PT-BR default, EN)

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run start
```
